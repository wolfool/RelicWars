## §1. 게임 컨셉

RelicWars는 **생존 + PvP + 보스전**이 결합된 마인크래프트 플러그인입니다.

- 서버에 **30개의 유물**(#030~#001)이 존재하며, 플레이어들은 유물을 획득하고 쟁탈합니다.
- **팀**(최대 2인)을 구성하여 유물을 모으고, 최종적으로 **왕좌의 핵(#000)**을 획득한 뒤 30분간 제단을 방어하면 시즌 승리입니다.

## §4. 유물 시스템

### 4.1 등급 체계

| 등급 | 번호 범위 | 색상 | 정신력 소모 |
|------|-----------|------|-------------|
| 1단계 (최하급) | #030~#025 | WHITE | 0 |
| 2단계 (하급) | #024~#020 | BLUE | 0 |
| 3단계 (중급) | #019~#011 | GREEN | 10 |
| 4단계 (상급) | #010~#006 | RED | 20 |
| 5단계 (최상급) | #005~#001 | GOLD | 30 |
| 특수 | #000 | PURPLE | — |

### 4.2 유물 규칙

- 인당 **최대 4개** 소지 (`relic.max-per-player: 4`)
- **Q키(드랍) 불가**, **상자 보관 불가** (인벤토리에만 보관)
- **우클릭**으로 능력 발동, 쿨타임 존재
- 봉인 유물에 발광 효과 부여 (`relic.seal-glow: true`)

### 4.3 봉인 시스템

바닥에 존재하는 유물은 **봉인 상태**입니다.
- 봉인 타이머가 0이 되면 **Claimable** 상태로 전환
- Claimable 유물을 **2초간 우클릭**하면 획득 (`relic.pickup-seconds: 2`)
- 봉인 시간은 출처에 따라 다름:
  - 다운 시 드랍: **30초** (`combat.downed-drop-seal-seconds`)
  - 사망 시 드랍: **30초** (`combat.death-drop-seal-seconds`)
  - 보스 드랍: **60초** (`combat.boss-drop-seal-seconds`)

## §5. 전투 시스템

> 코드 출처: `CombatManager.java`, `CombatListener.java`

### 5.1 다운 상태

치명상을 입으면 즉사하지 않고 **다운 상태**로 전환됩니다.

**다운 진입 시:**
1. `#005 불멸의 심장` 패시브 체크 (발동 시 다운 스킵)
2. HP 20으로 설정 + 디버프 적용: 둔화5(∞), 점프부스트128(점프불가), 실명(∞)
3. 정신력 **0으로 리셋**
4. `#017 왜곡의 닻` 체크 (50블록 내 닻이 있으면 텔레포트)
5. **가장 낮은 번호(최강) 유물 1개** 즉시 봉인 드랍 (30초 봉인)
6. **2초 무적** (`combat.downed-invincibility-seconds: 2`)
7. **환경 면역** — 화염/용암/낙하/익사 무시 (`combat.downed-environmental-immunity: true`)
8. **60초 자동 사망 타이머** 시작 (잔여 10초부터 심장박동 효과음)

### 5.2 확킬 (처형)

- 다운된 적을 **20회 공격** → `executePlayer()` 호출 (`combat.execute-hits: 20`)
- **추가 유물 드랍 없음** (다운 시 드랍된 1개만 존재)
- 스폰 지점으로 이동

### 5.3 강탈

- 다운된 적에게 **웅크린 채 우클릭** → 유물 추가 드랍
- 소지 유물 개수에 비례한 드랍:

| 피해자 소지 유물 | 추가 드랍 수 |
|-----------------|-------------|
| 0~1개 | 0개 |
| 2~3개 | 1개 |
| 4개 | 2개 |
| 5개 이상 | 3개 |

- 드랍 선택 순서: **내림차순** (높은 번호 = 약한 유물부터 드랍)
- 드랍된 유물은 **봉인 상태**로 스폰 (30초)
- 다운 1회당 최대 강탈 횟수: **3회** (`combat.max-relic-steals: 3`)

### 5.4 자동 사망

- 60초 경과 시 `killPlayer()` 호출
- **강탈 규칙에 따른 추가 드랍 발생**

### 5.5 구조

- 같은 팀원이 **8초간 우클릭** (`combat.revive-seconds: 8`)
- 회복 체력: **6 HP (3하트)** (`combat.revive-health: 6`)
- 특수 유물에 의한 단축:
  - `#025 최후의 봉합`: 구조 시간 **2초**
  - `#007 파수꾼의 돔`: 돔 내부 구조 시간 **3초**

### 5.6 랜뽑 방지

- **전투 태그 15초** (`combat.combat-tag-seconds: 15`)
- 전투 태그 중 접속 종료 → 즉시 다운 + `killPlayer()` (이중 페널티)
- 다운 중 접속 종료 → 즉시 `killPlayer()`

### 5.7 보이드 사망

- 모든 보호(#005, 다운 무적, 환경면역) **완전 무시**
- 즉시 `killPlayer()` + `cleanupPlayer()`

### 5.8 특수 유물 전투 상호작용

| 유물 | 상호작용 |
|------|----------|
| #005 불멸의 심장 | 다운 전 패시브 발동: HP 100% 회복 + 넉백 + 저항2 8초 |
| #002 탐욕의 적출자 | 다운된 적 우클릭 0.5초 → 모든 유물 자연 드랍(봉인X) + 즉사 |
| #017 왜곡의 닻 | 다운 시 50블록 내 닻으로 텔레포트 (1회용) |
| #025 최후의 봉합 | 구조 시간 2초 + 도주 버프 (이속2 10초 + 저항1 5초) |
| #007 파수꾼의 돔 | 돔 내부 구조 시간 3초 |

## §8. 이벤트 시스템

### 8.1 핏빛 달

- 매 밤 **15% 확률** 발생 (`event.blood-moon.chance-per-night: 0.15`)
- **3분 간격** 브로드캐스트 (`event.blood-moon.broadcast-interval-minutes: 3`)
- 유물 보유자의 대략 좌표 공개 (**100블록 흐림**, `event.blood-moon.coordinate-blur: 100`)

### 8.2 잊혀진 유물

바닥에 30분 이상 방치된 유물은 단계적으로 좌표 힌트가 공개됩니다:

| 경과 시간 | 공개 수준 |
|-----------|-----------|
| 30분 | 방향만 |
| 60분 | 1000블록 흐림 |
| 90분 | 500블록 흐림 |
| 120분 | 50블록 + Y좌표 흐림 |

## §10. 엔딩 조건

1. 전체 30개 유물이 서버에서 1번 이상 스폰 (`ending.required-all-spawned: true`)
2. 1팀이 10개 이상 보유 (`ending.required-team-relics-to-summon: 10`)
3. → 엔딩 제단 점령전 시작 (월드 스폰 지점)
4. → **오버워치 방식 점령전** — BossBar 게이지 0~100%
   - 아군만 반경 안 → 게이지 상승 (인원 비례)
   - 양쪽 다 있음 → **CONTESTED!** (완전 정지)
   - 적만 있음 → 게이지 감소 (인원 비례)
   - 비어있음 → 느린 감소
5. → 게이지 100% 도달 시 **시즌 승리**
6. 관리자 명령어: `/relic altar start|stop|status`
